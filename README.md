# GK Quiz App(Tkinter)
A Python-based GK Quiz App built using Tkinter. The app supports multiple topics, load questions from a JSON file, and displays the final score after completing the quiz.
## Technologies Used
-Python
-Tkinter
-JSON
## Project Structure
### Main application
-quiz_app.py
### Quiz questions
-quiz_data.json
